# ADR 0005: Trusted Source Manual Entry

**Status**: Accepted.

CompeteHub will treat administrator manual entry from trusted sources as the current-version way to create competition records, with source identity, source link, structured fields, and review before publication. Semi-automated collection may later create candidate competition records, but it must not bypass human review or become an automatic publishing pipeline without a new decision record because that would change data quality responsibility, audit requirements, failure handling, and trust assumptions across the product.
