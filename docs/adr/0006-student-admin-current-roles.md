# ADR 0006: Student and Admin Current Roles

**Status**: Accepted.

CompeteHub will implement students and administrators as the current formal user roles. Teachers, teaching secretaries, competition organizers, and student organizations remain auxiliary stakeholders that may influence requirements, but they do not receive independent workspaces, navigation, RBAC branches, or data-permission models in the current version. Promoting any auxiliary stakeholder into a formal role requires a new decision because it changes product scope, authorization design, administration workflows, and roadmap sequencing.
